What is the Document Object Model (DOM)?

Document Object Model(DOM) represents the structure of html document. It allows JavaScript to access and manipulate the content of a web page. It is a tree-like structure(parent-child).

How do you select an element from the DOM?

We can select an element from the DOM using some methods such as getElementById, getElementByClass, getElementByTagName, querySelector, querySelectorAll etc. getElementById method takes one parameter which is the "id" of the targeted element. getElementByClass takes a className and return all the elements which have that class. querySelector can take class, id or tagname and return only the first matching element. querySelectorAll returns all the elements matches the condition.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique to handle events efficiently. Instead of adding multiple event listeners to many elements, you can add just one listener to a parent element and let it handle events for its children. This technique can make our code clean and efficient.

How do you manipulate an element's attributes and styles using the DOM?

I can manipulate the attributes and styles of an element using the DOM. first of all I need to select the element. Then I can use the setAttribute, removeAttribute methods and styles property to change the css styles.
const myBtn = document.getElementById('my-btn');
myBtn.setAttributes('href', 'https://www.google.com');
myBtn.style.backgroundColor = 'red';
